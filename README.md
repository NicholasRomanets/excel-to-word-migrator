## About
excel-to-word-migrator reads rows from *.csv files (Excel workbooks) and generates Word (DOCX) using a DOCX template populated with table. It is useful for automating:
- Bulk migration of tabular data into formatted Word documents;
- Integrations where a spreadsheet is the source of truth and Word is the output format.

## Features
- Read .csv file;
- Use DOCX templates with table that map to *csv file rows.

## Prerequisites
- PyCharm >= 2025.2.1.1;
- Python >= 3.13.3.
